Text Formatting

HTML tags are the fundamental elements of HTML used for defining the structure of the document. These are letters or words enclosed by angle brackets (< and >). Usually, most of the HTML tags contain an opening and a closing tag. Each tag has a different meaning, and the browser reads the tags and displays the contents enclosed by them accordingly. For example, if we wrap any text in the paragraph tag, the browser displays it as a separate paragraph.This is a paragraph.
